# Docker alias and functions

Useful command line alias for [Docker](https://www.docker.io/)

## Install

In coreos

```coreos
cd && mv .bashrc .bashrc_old && curl -fsSL https://raw.github.com/agend07/docker-alias/master/zshrc > ~/.bashrc && source ~/.bashrc
```
